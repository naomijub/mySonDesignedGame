# mySonDesignedGame
##Instructions
### Moves:
arrows move the character.
### How to look to the other side without moving:
j - look left, i - look up, k - look down, l - look right;
### Selecting Weapon:
a - unselect weapons, s - select sword, d - select pickaxe;
### Attack:
space bar
#
##It is necessary Java 8 to play
